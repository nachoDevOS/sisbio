## Trabajo 2 — Integración con el microservicio (Python/FastAPI)

**Commit:** `(histórico)` — `feat: cliente del microservicio y acciones de equipo`

### Problema
Laravel no debe hablar por TCP directo con los equipos. Se necesitaba un único
punto de contacto que consulte estado y marcaciones vía HTTP.

### Archivos modificados
| Archivo | Cambio |
|---|---|
| `app/Services/DeviceService.php` | Cliente HTTP autenticado (`info`, `users`, `attendance`). |
| `app/Exceptions/DeviceServiceException.php` | Errores del microservicio en mensajes claros. |
| Pantalla de Equipos | Acciones "Probar conexión" y "Ver marcaciones". |

### Solución
`DeviceService` traduce cada operación a una petición HTTP con token
`X-Auth-Token`. "Probar conexión" guarda estado en línea, algoritmo y hora;
"Ver marcaciones" muestra las marcaciones leídas en vivo del equipo.
